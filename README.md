# ScamGuard AI
Objective: Build an LLM application for detecting scam messages.

LLM Used: flash-lite

Steps to use repository

# Architecture used
Facade pattern
Curtain Architecture


# Steps to run the project
1. Clone the repository
2. Install the requirements
3. Run the streamlit app
4. Open the streamlit app in the browser
5. Enter the message to be checked for scam
6. Click on the "Check" button
7. The result will be displayed on the screen

# Steps to run the project in docker
1. Clone the repository
2. Build the docker image
3. Run the docker container
4. Open the streamlit app in the browser
5. Enter the message to be checked for scam
6. Click on the "Check" button
7. The result will be displayed on the screen
